# 3D трассировочная карта для VS Code

Это версия без Jupyter и `ipywidgets`: управление работает прямо в браузере, а карту можно вращать мышью как 3D-объект.

## Запуск

1. Откройте папку `prog/vscode_3d_map` в Visual Studio Code.
2. Откройте `index.html`.
3. Запустите через расширение Live Server или просто откройте файл в браузере.
